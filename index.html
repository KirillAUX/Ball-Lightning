Инструкция по установке

Для Windows:

  1. Скачать архив нужной версии
  2. Распаковать
  3. И запустить игру

Для Mac Os:

  1. Убедитесь, что Java установлена: Откройте терминал и введите: java -version
  2. Если Java не установлена, скачайте с официального сайта или используйте Homebrew: brew install openjdk@21
  3. Перейдите в папку с вашим balllightning.jar: cd /путь/к/папке
  4. Запустите игру командой: java -jar balllightning.jar

Для Linux:

  1. Откройте терминал и введите: java -version
  2. Если Java не установлена, введите: sudo apt update && sudo apt install openjdk-21-jre
  3. Перейдите в папку с игрой: cd /путь/к/папке
  4. Запустите игру: java -jar balllightning.jar
