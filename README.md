# Ball-Lightning
🎮 ОПИСАНИЕ:
2D платформер с элементами шутера. Сражайтесь с врагами,
собирайте усиления, используйте супер-атаки!

🚀 ЗАПУСК ИГРЫ:
Запустите файл "Ball Lightning.exe"

⚙️ СИСТЕМНЫЕ ТРЕБОВАНИЯ:
- Windows 7/8/10/11/Linux/macOS
- Java 21 или выше (скачать: https://adoptium.net/)
- 1 ГБ оперативной памяти
- DirectX 9 или выше
- Место на диске: 50 МБ

🎯 Управление персонажем
- ↔ Движение влево/вправо: Стрелки или A/D
- ⬆ Прыжок: Пробел, W или Стрелка вверх
- ⚔ Обычная атака: E (бесконечные пули)
- 💥 Супер-атака: Q (ограниченный запас)
- ⏸ Пауза: ESC или P


🆘 ЕСЛИ ИГРА НЕ ЗАПУСКАЕТСЯ:
1. Установите Java 21: https://adoptium.net/
2. Запустите через командную строку:
   java -jar balllightning.jar
3. Или используйте StartGame.bat

📞 ПОДДЕРЖКА:
Если возникли проблемы, проверьте:
- Установлена ли Java 21+
- Наличие папок sounds/ и textures/ рядом с EXE
- Достаточно ли свободной памяти

🎵 Особенности:
- Полная звуковая система с MP3
- Несколько типов врагов
- Система усилений
- Меню паузы и настроек
